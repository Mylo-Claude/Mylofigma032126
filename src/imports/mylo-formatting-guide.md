Links and References

Links are structural markers that connect to external resources. For example, visit the Mylo documentation for more details. Links can also be combined with bold or italic formatting.

Superscript and Subscript

Scientific notation requires superscript, like E = mc2, and chemical formulas need subscript, like H2O. Footnote references1 also use superscript.

Template Governance

All structures and markers shown in this guide are governed by templates. The Editor captures structural intent using drafting typography for clarity. The Preview applies template rules to produce governed, brand-accurate rendering.

Switch between templates in the Preview panel to see how the same structural content renders differently under each template's governance rules.
